# JizoProject
奈良市の道端の地蔵堂や小祠、石仏などをオープンデータ化するプロジェクトです。データストアとしては[Wikimedia](https://commons.wikimedia.org/wiki/Category:Wayside_Place_of_Worship_in_Nara)に蓄積しつつ、半自動でGeoJson化してこちらに随時公開します。
